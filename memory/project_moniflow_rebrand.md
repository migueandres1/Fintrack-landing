The project was rebranded from FinTrack to MoniFlow — a personal finance platform for Centroamérica y el Caribe (Guatemala, Honduras, El Salvador, Costa Rica, Nicaragua, Dominican Republic, Panama, Jamaica).

**Why:** User provided full MoniFlow brand book (Design System v2.0), research foundation, and strategy/planning docs. The landing page was fully rebuilt to match the brand identity.

**Key design decisions:**
- Palette: Selva Nocturna (`#0b1712` dark) & Agua Caribe (`#00b894` accent)
- Typography: Cormorant Garamond (H1/H2 via `font-display`), DM Sans (body via `font-sans`), DM Mono (numbers via `font-mono`)
- Dark hero sections on `#0b1712`, light content sections on `#f0f5f3`
- Footer: `#060e0a` (darker than hero)
- Stack kept: React + Vite + Tailwind CSS frontend, Express + Stripe backend

**Pricing (as specified by user on 2026-03-31):**
- Free: $0 forever
- Pro: $2.99/mo or $29.90/yr
- Familia: $3.99/mo or $39.90/yr (up to 5 users)
- All paid plans: 30-day free trial

**App URL:** `https://app.moniflow.com` (changed from app.fintrack.com)

**Landing sections (in order):** Navbar → Hero (dark, split 55/45) → Banks strip → Pain Points (light) → Features (dark) → Remesas differentiator (dark) → How It Works (light) → Testimonials (light) → Competitor comparison (dark) → Pricing (dark) → Final CTA (dark) → Footer

**How to apply:** All new features or UI changes must follow the MoniFlow Design System v2.0. Colors, fonts, and tone must match the brand book. Brand doc is the source of truth.
